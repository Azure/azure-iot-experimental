Troubleshooting tips to make your Chirp with Azure IoT Central experience awesome!

A. Adjust the volume of the speakers on your computer. The MXChip microphone is very sensitive if the volume is too high the sound data will be clipped and result in data corruption. Inversly make sure the volume is not too quiet and not getting picked up by the microphone. Whilst the sound data can be heared quite easily against background white noise, loud voices or music in the same room may cause data corruption.

A. Go to your Azure IoT Central application and login. In device explorer click the device you wish to connect. In the top right corner click "Connect this device". A popup will show your connection string and you can click the copy button to copy it to the clipboard.

A. There are two options to put the firmware for this example onto the MXChip device. The first option is to use the the pre-built firmware that can be found in the directory brebuilt_firmware. The chirpFirmware.bin file inside the directory can be dragged and dropped onto the AZ3166 share that should be shown on your computer when the MXChip is connected by USB. The second option is to use Visual Studio Code and the Arduino plugin to build the firmware from the source in the firmware directory and let the Arduino plugin flash the firmware onto the device. Instructions for installing the Arduino plugin and the MXChip SDK can be found here.

A. To reset the WiFi and device connection string hold down the A and B button until you see the display show "Resetting ..." (Note: the message pops up and disappears pretty quickly so keep an eye out for it). Release the buttons and the device will reboot back to listening mode and you can reconfigure it from the configuration web page.

A. If you experience a problem and cannot find the answer on this page then enter an issue on the issues page in the Github repo. We will then try and resolve the issue and post an answer on the issues page. If the issue has been reported multiple times or we feel it is a significant problem we will add the question and answer to this troubleshooting guide.
